## 1: install CRNA
$ npm install -g create-react-native-app

## 2: create a project
$ create-react-native-app my-app
$ cd my-app/

## 3: replace all files

## 4: npm installing
$ npm install
$ npm install native-base --save
$ npm install @expo/vector-icons --save
$ npm install --save react-native-router-flux

## 5: run
$ npm start
